# WA Intelligence Platform

This is a monorepo for the WA Intelligence Platform which includes frontend, backend, and Supabase configurations along with Docker setup.

## Structure

```plaintext
WA-Intelligence-Platform/
├── frontend/
│   └── (Frontend project files)
├── backend/
│   └── (Backend project files)
├── supabase/
│   └── (Supabase project files)
└── docker-compose.yml
```

## Getting Started

### Prerequisites

- Docker
- Node.js (for frontend and backend)
- Supabase CLI (for Supabase project)

### Installation

1. Clone the repository
2. Navigate to the frontend or backend directory and install dependencies
3. Start the application using Docker or individually.
